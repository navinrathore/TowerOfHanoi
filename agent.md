# MyTowerOfHanoi

## Project Overview
THe project implements the classic Tower Of Hanoi problem with different approaches to solve it. 

## Stack
- Framework: FastAPI
- Language: Python
- Styling: Tailwind CSS
- Database: SQLite
- Hosting: Self-hosted
- Testing: pytest
