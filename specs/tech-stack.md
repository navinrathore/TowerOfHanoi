# Tech Stack

MyTowerOfHanoi is a Python-based FastAPI web application. The frontend uses server-side Jinja2 HTML templates styled with Tailwind CSS, augmented by vanilla JavaScript for client-side interactivity, state management, and animations.

## Core Stack

| Layer | Choice | Rationale |
|---|---|---|
| Language | Python (3.10+) | Standard language for Deep Learning / RL; excellent web backend libraries. |
| Web Framework | **FastAPI** | Extremely fast, native async support, autogenerated OpenAPI docs, and easy integration with Jinja2 templates. |
| Templating | Jinja2 | Clean, robust server-side rendering for HTML templates. |
| CSS | **Tailwind CSS** (via CDN) | Kept as simple as possible. No build step/node dependency. Served via CDN script in main layout. |
| JS Interactivity | Vanilla JavaScript | Simple DOM manipulation, drag-and-drop or click-to-move animations for the disks, and Fetch API calls for the solvers. |
| Database | **SQLite** (embedded) | Single-file database, zero setup, perfect for lightweight local run stats and model logs. |
| ORM / DB Access | **SQLAlchemy** or SQLModel | Pythonic way to handle database queries and models with type safety. |
| Testing | **pytest** | Python's standard, powerful testing framework. |

## Tooling & Quality

- **Uvicorn**: ASGI web server to run FastAPI.
- **Ruff / Black**: Fast linting and formatting.
- **Mypy**: Strict type-checking tool for Python.
- **Git**: Version control.

## Reinforcement Learning Layer

- **Q-learning Agent**: Implemented in pure Python using custom numpy-like or pure dict/table data structures. This avoids large deep learning framework overhead (like PyTorch or TensorFlow) while cleanly showing reinforcement learning concepts (states, actions, Q-values, exploitation vs exploration, rewards).

## Directory Structure

```text
/deepLearning/TowerOfHanoi/
├── specs/                 # SDD Specifications
├── static/                # Static assets (images, custom CSS, JS)
├── templates/             # Jinja2 HTML templates
├── database.py            # SQLite connection and session setup
├── models.py              # SQLAlchemy/SQLModel definitions
├── main.py                # FastAPI entry point
├── game.py                # Tower of Hanoi state engine
├── solvers/               # Solver implementations
│   ├── __init__.py
│   ├── classic.py         # Recursive and Iterative solvers
│   ├── search.py          # A* search solver
│   └── qlearning.py       # Q-learning agent training & execution
├── tests/                 # pytest test suite
├── requirements.txt       # Python dependencies
└── agent.md               # Project metadata / CLAUDE.md
```
